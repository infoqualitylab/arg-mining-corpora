# AMResources
This repository contains the source code and data for a static website containing a detailed list of datasets for argumentation mining and their associated papers. Datasets are organised to maintain their inheritence relationships with other datasets. Full details can be found in our upcoming Argument Mining 2016 workshop paper "AMResources: Cataloging Argument Mining Datasets". This website also serves as an interactive and growing appendix to the forthcoming "Argument Mining, 2nd Edition" authored by Manfred Stede, Jodi Schneider, and Henning Wachsmuth.

# Updating the Site
This website will be a maintained resource, and encourages contributions from the community. Data from the site can be downloaded in an excel format from the deployed [page](TODOlinktopageshost), or the full JSON for the [papers](src/data/entries/papers.json) and [datasets](src/data/entries/datasets.json) can be downloaded. The site is intended to document datasets, therefore we will not add new papers to the site unless they introduce or annotate datasets, and we will not add any datasets to the site that do not come from an associated paper. If you wish to add a new datasets/paper to the site see below.

## Submitting a Formatted Issue
Fill out the form on the website under the "Submit Dataset" tab. This will submit an issue with your request formatted into a small JSON file. Please provide as much detail as possible and draw attention to what you think the most salient parts of the dataset are. A link to an associated paper is essential. We will then read the paper and provide a handwritten description for the site. We will never use AI to generate dataset descriptions.

## Raising Other Issues
Please raise any other issues in the issues tab. This includes errors with the site and any potential mistakes in dataset descriptions. In general we try to provide a broad summary of datasets, highlighting what we think are the most relevant features, but we are open to any feedback.
